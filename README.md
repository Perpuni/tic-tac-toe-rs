# tic-tac-toe-rs
Simple tic tac toe game in terminal

````
Welcome to CLI tic tac toe!
Type a row and column (both between 1 and 3)
It's O move

 -  -  -     1 1   1 2   1 3
 -  -  -     2 1   2 2   2 3
 -  -  -     3 1   3 2   3 3
````

To install
````
cargo run

# linux
xdg-open tri.png

# macos
open tri.png
````
